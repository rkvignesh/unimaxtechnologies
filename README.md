## Getting Started

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start
```

The app will be available at `http://localhost:4200/`

### Build for Production

```bash
npm run build
```

## Customization

### Colors
Edit the CSS variables in `src/styles.css`:

```css
:root {
  --color-primary: #0071E3;
  --color-text: #1D1D1F;
  /* ... more variables */
}
```

### Content
Edit `src/app/app.component.ts` to modify:
- Services array
- Products array
- Features array

### Images
Replace emoji placeholders with actual images in HTML or component data
